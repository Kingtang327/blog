# 介绍

## 什么是JVM

JVM即Java Virtual Machine（Java虚拟机）.

JVM是一种用于计算设备的规范，它是一个**虚拟的计算机**.

JVM是一种规范,意味着必定有很多种规范的实现,最常用的就是HotSpot.

既然是虚拟的计算机,它在架构上肯定和真正的计算机是类似的.

这一点非常重要,对于我们后续理解JVM有很重要的意义!
